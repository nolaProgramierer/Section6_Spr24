Application Information

This is the index page for the piano inventory application. It is a hybrid application utilizing the Django, Django Rest and the React JavaScript frameworks. Any user can view a list of pianos in the inventory. By registering a logged-in user may create, edit or delete a piano. The link below takes a user to the inventory page which is rendered by both Django and React.
